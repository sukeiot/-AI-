# 速科物联 AI 系统 V1

速科物联 AI 系统 V1 是一套基于 PHP + MySQL 构建的轻量级人工智能对话平台，集成了 DeepSeek AI 接口，配备实名验证机制及基础用户管理后台。系统界面采用简约风格，便于快速部署与使用。

## 系统功能

- 接入 DeepSeek AI 平台（通过 API 调用）
- 前端界面简洁，适配常见浏览器
- 用户注册、登录、实名、AI 对话集于一体
- 强制用户二要素实名认证（姓名+身份证号）
- 后台可管理用户账户
- 适合自建平台或测试部署
